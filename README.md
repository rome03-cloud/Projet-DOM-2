# Color Changer Interactif 🎨

Un projet web simple et élégant qui permet de changer dynamiquement la couleur d'une boîte à l'aide d'un bouton. Ce projet a été réalisé pour pratiquer la manipulation du DOM, la gestion des événements et l'intégration HTML/CSS/JS.

## 🚀 Fonctionnalités
- Génération de couleurs hexadécimales aléatoires.
- Interface utilisateur centrée et moderne.
- Transitions fluides lors du changement de couleur.
- Design responsive.

## 🛠️ Technologies utilisées
- **HTML5** : Structure de la page.
- **CSS3** : Mise en forme et animations (Flexbox, transitions).
- **JavaScript (ES6)** : Logique interactive et manipulation du DOM.
